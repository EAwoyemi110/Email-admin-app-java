# Email Admin App (Java)

A simple Java console application that simulates an internal employee email administration tool.  
The program creates a new employee account, assigns a department, generates a random password, and builds a company email address automatically.

## Overview

This project was built to practice object-oriented programming in Java through a realistic business-style use case. It focuses on class design, encapsulation, constructors, private helper methods, user input handling, and string manipulation.

The application walks through the process of creating an employee email account by:
- collecting the employee’s first and last name
- prompting for a department selection
- generating a random password
- constructing a company email address based on the employee’s information

## Features

- Object-oriented class design
- Encapsulated employee account data
- Department selection through console input
- Random password generation
- Automatic company email creation
- Beginner-friendly Java project structure

## Example Workflow

1. User creates a new employee
2. Program prompts for a department code
3. Program assigns the department
4. Program generates a random password
5. Program creates the employee’s business email address
